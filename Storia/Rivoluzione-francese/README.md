# La Rivoluzione francese — PWA didattica

A cura di gbprof e Libera.

## Uso su PC
Apri `index.html` nel browser per navigare. Per avere pieno comportamento PWA/offline, avvia un piccolo server locale dalla cartella del progetto:

```bash
python -m http.server 8080
```

Poi apri `http://localhost:8080`.

## Uso su GitHub Pages
Carica tutti i file di questa cartella in un repository GitHub e attiva GitHub Pages sul branch principale.

## Contenuti
- Index interattivo conforme alla copertina approvata.
- 4 lezioni principali in PDF e mappa.
- Approfondimenti in PDF e schema.
- Galleria mappe.
- Biografie.
- Timeline.
- Quiz finale.
- Manifest e service worker per uso PWA.
