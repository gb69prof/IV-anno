# Corpus gbprof · Illuminismo — sito prova

## Avvio rapido
Aprire `index.html` con un browser moderno. Per installazione PWA e uso offline completo è preferibile servire la cartella tramite un piccolo server locale.

Esempio:
```bash
python -m http.server 8000
```
Poi aprire `http://localhost:8000`.
